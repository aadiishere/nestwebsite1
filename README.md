# Nest Digital Website

## Overview

This project is a **responsive corporate website** developed during my **Frontend Development Internship at Nest Digital**.
The website was built using **HTML, CSS, JavaScript, and Bootstrap** to understand real-world frontend development practices and component-based UI design.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap

---

## Features

* Responsive layout using Bootstrap grid system
* Navigation bar with multiple sections
* Image carousel and cards
* Accordion and interactive UI components
* Clean and professional design

---

## Project Purpose

The objective of this project was to:

* Apply frontend fundamentals in a real internship environment
* Learn Bootstrap components and layout system
* Improve UI structuring and responsiveness
* Gain hands-on experience building a corporate-style website
